# 3XO (color-ground)

### 서비스 링크

- Playstore : https://play.google.com/store/apps/details?id=com.colorground
- Appstore : https://apps.apple.com/kr/app/3xo/id1544711479

### 개발환경

### 개요
사진에서 대표색 3개를 추출해 색을통해 사진을 기억하는 iOS 어플리케이션.
